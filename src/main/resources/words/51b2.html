    基本释义

    [ chōng ]

    1.用水或酒浇注，水撞击：～茶。～剂。～洗。～荡。

    2.向上钻，直上：～腾。～入云霄。

    3.破解不祥：～喜。

    4.空虚，谦虚：“大盈若～，其用不穷”。～挹（yì）。谦～。

    5.幼小：～昧。～弱。～龄。

    6.方言，山区的平地：韶山～。

    7.通行的大路，重要的地方：要～。首当其～。

    8.不顾一切，一直向前：～锋。横～直撞。

    9.猛烈地撞击：～力。～突。

    10.收支账目互相抵销：～账。

    11.情感强烈：～动。兴（xìng）～～。

    12.太阳系中，除水星和金星外，其余的某一个行星进行到与地球、太阳成一条直线而地球正处在这个行星与太阳之间的位置时称“冲”。

    [ chòng ]

    1.对着，向着：～着太阳。

    2.凭，根据：～你这么好学，一定能有成就。

    3.猛烈；强烈：～劲。这酒真～。

    4.一种使金属板成形或在金属板上打孔的专用设备称“冲床”。

    5.在冲床上加工金属工件：～压。

    详细释义

    1. 冲 [chòng]2. 冲 [chōng]

    冲 [chòng]

    〈形〉

    1. 猛劲大 。

      如:有股冲劲儿;水流得真冲

    2. 气味浓烈刺鼻 。

      如:酒味儿冲

    〈动〉

    1. 用冲床进行金属加工 。

      如:冲床;冲模

    〈介〉

    1. 对着;向 。

      如:冲着(对着,向着);大门冲北开

    2. 凭;根据 。

      如:就冲着他,我也不得不出面解围

    3. 另见 chōng

    冲 [chōng]

    〈动〉

    1. (形声。从水,中声。本义:向上涌流)

    2. 同本义

      冲溶沅瀁。——木华《海赋》

    3. 直上

      又如:冲天(直上天空);冲汉(直上霄汉)

    4. 以水、酒等浇注调制 。

      如:冲鸡蛋;用酒冲服;冲茶

    5. [会计]收支账目互相抵消,或两户应支付的款项互相抵消

      幸而后手里渐渐翻过来了,除了冲账的,反赢了好些——《红楼梦》

      又如:红字冲销

    6. 使曝光过的照相材料显影 。

      如:冲胶卷

    7. 用水或其他液体冲洗 。

      如:把盘子冲一冲;冲厕所;秧苗给大水冲了

    〈形〉

    1. 深远 。

      如:冲迈(高远);冲音(精深玄妙之言)

    2. 通“盅”。空虚

      又如:冲室(虚室,空室);冲寞(虚寂恬静)

    3. 平和;谦虚

      又如:冲融(融和充实);冲和(性情谦虚和平)

    4. 淡泊 。

      如:冲虚(淡泊虚静);冲衿(谦和淡泊的胸襟);冲静(恬静淡泊)

    5. 通“僮(童)”。幼小

      又如:冲弱(年幼,幼稚);冲年(幼年);冲幼(年幼);冲眇(幼小)

    〈名〉

    1. (形声。从行,重声。“行”,意思是路。本义:交通要道)

    2. 同本义

      及冲,以戈击之。——《左传·昭公元年》。注:“要道。”

      冲风起兮水横波。——《楚辞·河伯》。注:“隧也。”

      太冲绝。——《素问·至真要论》。注;“肝脉气也。”

      又如:冲衢(交通要道);冲途(要道;大路);冲境(冲要之地;交通发达地区)

    3. 古战车名。用以攻城

      与尔临冲。——《诗·大雅·皇矣》

      主人焚冲。——《左传·定公八年》

      折冲万里。——《淮南子·说山》

      隆冲以攻。——《淮南子·泛论》

      又如:冲车(古兵车名。用以冲城攻坚);冲隆(古兵车)

    4. 山区的平地 。

      如:翻过山有一个很大的冲

    5. [天文]∶一个天体与另一个天体在天空中的相反方向或距角等于180度的一种行星动态 。

      如:大冲;冲日(外行星运行到与地球太阳成一直线且与太阳距角等于180°)

    〈动〉

    1. 冲击;冲撞

      稍近,益狎,荡倚冲冒。——柳宗元《黔之驴》

      此鸟不飞则已,一飞冲天;不鸣则已,一鸣惊人。——《史记·滑稽列传》

      怒发上冲冠。——《史记·廉颇蔺相如列传》

      又如:冲激(冒犯);冲打(冲击攻打);冲阵(冲击敌方战阵)

    2. 直朝某一方向而去 。

      如:冲蒙(奔驰于草丛);冲斥(犹充斥)

    3. 破除;毁坏 。

      如:冲波(冲破波浪);冲溢(溃决泛滥)

    4. 冒。多指不顾危险或恶劣环境而向前行进 。

      如:冲雨(冒雨);冲炎(冒着炎热)

    5. 动 。

      如:冲心(触动心境;动心);冲心钱(数目足以引人动心的钱)

    6. 冲刷;灌注 。

      如:冲注(水流冲击)

    7. 另见 chòng

    百科释义

    报错
    冲，chong： 1. 从氵（水）从中，涌摇也。1. 冲洗、冲刷、冲凉。2. 从行从重，以有重量的行动，产生出的力量：当也，向也，突也。一飞冲天、冲击、冲突、冲撞、冲破。简体字的”冲“为从冫从中，其中从冫为无义简化。冲是位置天文学的一个名词，是从一个选定的特定天体 （通常是地球） 观察另一个天体与参考天体 （通常是太阳） 的位置时，三者在一条直线上，但是在参考天体的另一边。明确的说，当一颗行星在冲的位置时，它与太阳的的黄经相差180°。查看百科
