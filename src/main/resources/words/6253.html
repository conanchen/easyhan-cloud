    基本释义

    [ dǎ ]

    1.击，敲，攻击：～击。殴～。～杀。

    2.放出，发出，注入，扎入：～炮。～雷。～信号。～电报。

    3.做，造：～首饰。～家具。

    4.拨动：～算盘。

    5.揭，破，凿开：～破。～井。

    6.举，提起：～灯笼。～起精神。

    7.涂抹，印，画：～蜡。～戳子。

    8.写出，开出：～证明。

    9.捆，扎：～包裹。

    10.合，结合：～伙。～成一片。

    11.获取，购取：～水。～鱼。

    12.除去：～消。～杈。

    13.定出，计算：～算。～腹稿。

    14.用，采用，使用：～比喻。

    15.玩，玩耍：～球。

    16.截，停，减，退：～住。～价儿。

    17.表示人体发出某种行为动作：～手势。

    18.进行某种活动，从事或担任某种工作：～交道。～短工。

    19.与某些动词结合，基本上保留原动词词义：～扫。～扰。

    20.自，从：～哪儿来？

    [ dá ]

    量词，指十二个：一～铅笔。

    详细释义

    1. 打 [dá]2. 打 [dǎ]

    打 [dá]

    〈量〉

    1. 12个构成的一组 。

      如:一打火柴;一打鸡蛋;一打铅笔

    2. 另见 dǎ

    打 [dǎ]

    〈动〉

    1. (形声。从手,丁声。本义:击)

    2. 同本义

      打,击也。——《说文新附》

      身世浮沉雨打萍。——宋· 文天祥《过零丁洋》

      又如:用棍打狗;打拍(敲击);打火刀(用来敲打火石的刀形铁片)

    3. 殴打;斗殴。亦指武术表演或戏剧中的武打 。

      如:打捶(打架);打孽(报仇,械斗);打把子(练武功)

    4. 攻打;进攻 。

      如:打援(攻打敌人的援军)

    5. 射击 。

      如:打冷枪;打炮

    6. 攻讦 [rake up sb.’s past and attack him]。

      如:打短(说别人短处)

    7. 发出 。

      如:打哨(吹口哨);打唤(呼唤)

    8. 猜,猜谜 。

      如:打一字;打一物;打虎(猜谜);打灯虎(猜灯谜)

    9. 围棋术语。即叫吃 。

      如:打劫(谓双方在一处可以交换吃一子的争夺战)

    10. 印上;拓印 。

      如:打本(即拓本);打郑方(盖公章);打印子(盖图章)

    11. 扎入;注入 。

      如:打针

    12. 往;去

      你打华州,须从这条路走。——《水浒传》

    13. 制造;建造

      九子不葬父,一女找荆棺。——《荆棺峡谵》

      又如:打船(造船)

    14. 砍;割

      只在左近人家佣工,替人家挑土打柴。——《儒林外史》

    15. 汲取;盛取 。

      如:打米(盛米,装米)

    16. 捕捉;猎取 。

      如:打捕(打猎捕鱼,渔猎)

    17. 买 。

      如:打酒;打票;打勾(购买)

    18. 振作 。

      如:打扳(振作);打迸(振作,焕发)

    19. 除去

      风流总被雨打风吹去。——宋· 辛弃疾《永遇乐·京口北固亭怀古》

      又如:打麦(用工具使麦子去皮或脱粒);打灭(打消;消除;消灭)

    20. 摇荡

      西江水阔 吴山远,却打船头向北行。—— 王建《扬州寻张籍不见》

    21. 收获 。

      如:打夹账(谓经手人从中索取好处费);打抹(谓没收额外所得酬金)

    22. 贬;退;减 。

      如:打扣(降低商品的原定价格;减少或者降低事物的数量、质量);打短(克扣)

    23. 评定 。

      如:打成分

    24. 相处;交结 [deal with;have contact with sb.]。

      如:打交待,打交道;打伙计(交朋友,恋爱);打影(结伴;合伙)

    25. 张、举、提。亦谓携带 。

      如:打旗

    26. 编织 。

      如:打草垫;打竹席;打领带;打个结;打发辫

    27. 转;绕 。

      如:打方旋(徘徊);打胡旋(兜圈子);打旋磨(盘旋;围绕着某物转);打转转(旋转,绕圈子);打独磨(盘旋;徘徊);打踅(打转。走江湖,跑码头)

    28. 把一物附着在他物上 。

      如:打胭脂,打粉;打背(裱糊布片)

    29. 谓用某种语言或腔调说话 。

      如:打牙配嘴(谓相互戏谑,闲扯说笑);打令(行酒令)

    30. 谓从事某种工作或做出某种行动 。

      如:打饼(做面食);打印子(借印子钱);打行(保镖、打手的行帮)

    31. 表示通过一定手段完成某种任务 。

      如:打铺;打介绍信;打轿;打花(采花;摘花)

    32. 与某些动词结合成复词,表示进行之意。

      如:打掠(用眼扫视);打勘(查问;拷问,审讯)

    〈介〉

    1. [口]∶从 ——带有北方方言色彩,普通话里一般用“从”,在单音方位词前只能用“从”。

      如:从南到北,从早到晚,从里到外

    2. 表示处所、时间、范围的起点。

      如:打这儿往东去;打县里回来;打明儿开始

    3. 表示经过的路线、场所。

      如:打水路走,三天可以到

    〈名〉

    1. 武术 。

      如:教打;练打。亦指京剧里指武打。

      如:唱、做、念、打

    2. 另见 dá
