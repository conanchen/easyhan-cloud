    基本释义

    [ shàn ]

    1.摇动生风取凉的用具：～子。～坠（扇柄下端的装饰物）。～面儿。～形。

    2.指板状或片状的屏：门～。隔～。

    3.量词，用于门窗等：一～门。

    [ shān ]

    1.摇动扇子或其它东西，使空气加速流动成风：用扇子～。

    2.鼓动别人去做不应该做的事：～动。～风点火。

    3.用手掌打：～耳光。

    详细释义

    1. 扇 [shàn]2. 扇 [shān]

    扇 [shàn]

    〈名〉

    1. (会意。从户从翅省。户,单扇门;从翅,像鸟的翅膀,可以张合。本义:门扇)

    2. 同本义

      扇,扉也。门两旁如羽翼也。——《说文》

      乃修阖扇。——《礼记·月令》。注:“用竹葦曰扇。”

      又如:扇板(门板);扇扉(门扇);窗扇;隔扇;火狐皮帽的耳扇

    3. 扇子

      箑,自关而西谓之扇。——《方言五》。按,即菨也。

      盖以杨门之扇。——《吕氏春秋》。注:“屏也。”

      有蒲葵扇五万。——《晋书·谢安传》

      又如:芭蕉扇;羽毛扇;蒲扇;电扇;扇市(以出售扇子等夏季用品为主的市集);扇拂(拂除灰尘的扇子);扇坠(悬于扇柄的饰物);扇月(圆月。以其如团扇,故称)

    4. 古代仪仗中障尘蔽日的用具,也叫“障扇”或“掌扇” 。

      如:扇莨( -làng)(古仪仗中伞的曲盖);扇伞(仪仗所用的长柄掌扇和伞盖)

    5. 毛巾、盖头之类织品

      候扇举而清叫,野闻声而应媒。——《文选·潘岳·射雉赋》

    〈量〉

    1. 用于门、窗等扁形器物

      旁开小窗,左右各四,共八扇。——明· 魏学洢《核舟记》

      又如:一扇门;两扇窗

    〈动〉

    1. 遮蔽

      榆性扇地,其阴下五谷不生。——贾思勰《齐民要术》

    2. 用同“骟”。割去马的睾丸或卵巢 。

      如:扇马(阉割过的马)

    3. 另见 shān

    扇 [shān]

    〈动〉

    1. 摇扇

      吹,扇助也。——《方言十二》

      艳妻扇方处。——《诗·小雅·十月之交》。毛本作煽。

      一扇息火,二扇生风,三扇下雨。——《西游记》

      又如:扇风(摇扇以生风);扇暍(传说周武王曾替中暑人扇风取凉);扇枕温衾(对父母老人极尽孝道之意)

    2. 以掌批面 。

      如:扇耳光;扇了几巴掌

    3. 施展

      宋官御史,坐行赇免,居林下,大扇威虐。——《聊斋志异》

    〈动〉

    1. 煽动、鼓动[他人做坏事]

      奸谄颇相扇构。——《晋书·谢安传》

      又如:扇构(煽动众人借机迫害);扇诱(煽惑引诱);扇惑(煽动蛊惑);扇聚(煽动纠合)

    2. 振翼欲飞 。

      如:扇动(指鸟禽飞虫鼓翅);扇翅(振翅)

    3. [风] 起,吹 。

      如:扇风(起风,鼓风)

    4. 宣扬;传播 。

      如:扇扬(宣扬,张扬);扇和(宣扬附和)

    5. 遮蔽 。

      如:叶死不扇(树叶枯死,不会遮阴)

    6. 泛指兴起、发生

      云雷方扇,鞠义旅以勤王。——《梁书》

    7. 扑打

      忽有皂雕,飞起直来台上扇妲己。——《武王伐纣平话》

    〈形〉

    1. 炽盛。通“煽”

      阎妻骄扇,日以不臧。——《汉书·谷永传》

      逮乎江左,此道弥扇。——《梁书·谢举传论》

      暨乎暮年,此风愈扇。——《北史》

    2. 另见 shàn
