    基本释义

    [ kē ]

    下巴颏儿，脸的最下部分，在两腮和嘴的下面。

    [ ké ]

    鸟名：红点～丨蓝点～。

    详细释义

    〈名〉

    1. 下颌下方或下颌支之间的表面,通称“下巴”

      颏,颐下也。——《玉篇》

      我手承颏肘拄座。——韩愈《记梦》

      又如:颏颐(下巴和腮帮子);颏颔(下巴);颏颊(脸面,面子)

    2. 另见 ké

    3. 頦 ké

    4. ——鸟名用字,根据鸟或哺乳动物颏下的一块异色羽毛或毛皮分类定名 。

      如:红点颏,蓝点颏

    5. 另见 kē
