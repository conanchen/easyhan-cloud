    基本释义

    [ héng ]

    1.跟地面平行的，与“竖”、“直”相对：～梁。～楣。～额。～幅。～批。～披（长条形横幅字画）。～匾。～标。～空。

    2.左右向的，跟目视方向垂直的，与“竖”、“直”、“纵”相对：～写。～排。～向。～斜。纵～。

    3.地理学上指东西向的：与“纵”相对：～贯东西。

    [ hèng ]

    1.凶暴，不讲理；蛮～。强～。～暴。～蛮。

    2.意外的，不寻常的：～财。～祸。～事。～死。

    详细释义

    1. 横 [hèng]2. 横 [héng]

    横 [hèng]

    〈形〉

    1. 假借为“犷”。横暴;放纵

      处士横议,以待我以横逆,横政之所出,横民之所止。——《孟子》。按,皆纵恣悍厉之谓。

      横行天下。——《荀子·修身》。注:“不顺理而行也。”

      东夷横畔。——扬雄《长杨赋》。注:“自纵也。”

      以威力相胁曰横。——《汉书·彭宠传》注

      又如:蛮横;横政(暴政);横侈(专横放纵);横虐(专横肆虐);横逆(横暴不顺理);横逞(放纵恣肆);横施(肆意施加);横议(恣肆批评);横杀(肆意杀戳)

    2. 意外,突然 。

      如:横生(意外发生,到处乱生);横起(意外的发生);横亡(横死);横夭(意外的早死)

    3. 额外 。

      如:横入(额外收入);横恩(额外的恩赏);横费(浪费;额外用度);横赐(额外的赏赐)

    4. 祸害

      谓为三横。——《世说新语·自新》

      三横唯余其一。

    5. 另见 héng

    横 [héng]

    〈名〉

    1. (形声。从木,黄声。本义:门框下部的横木)

    2. 同本义

      横,闌木也。——《说文》。字亦作桁。

      以横为之。——《诗·陈风·横门之下》

      又如:横轼(古代车前供人凭倚的横木)

    3. 侧,旁边

      时横隤以阳遂。——王褒《洞箫赋》。注:“旁决貌。”

      又如:横头(正面两侧的位置,或长方形物体较短两侧的位置);横屋(正屋前两侧的房屋);横道(旁道);横逸(犹旁出,旁行)

    4. 古代负有督察记录权责的官员,可以入人于罪者

      下有五横,以揆其官。——《管子·君臣》

    5. 学堂 。

      如:横舍;横塾

    6. 汉字由左至右的笔形“一” 。

      如:王字是三横一竖

    7. 姓

    〈形〉

    1. 地理上指东西向。与“纵”相对

      横田其亩。——《韩诗南山》。传:“东西曰横。”

      不别横与纵。——《楚辞·沈江》。注:“纬曰横。”

      纵横间之。——《淮南子·览冥》

      坐横弓。——《仪礼·大射仪》。注:“南踣弓也。"

      曰从与横。——《太玄卷·七玄莹》。注:“东西曰横。”

      又如:横织(纵横交织);横厉(纵横凌厉,形容气势强盛);横驱(纵横驰驱);横驱别骛(纵横奔驰)

    2. 跟地面平行的

      横柯上蔽。——吴均《与朱元思书》

      又如:横枝(主干上横长的树枝,比喻正宗以外旁衍的支派);横軃(横躺而睡着);横堑(与前方平行的濠沟);横卷(左右两端有轴的横幅书画)

    3. 自左到右或自右到左

      横执之。——三国魏· 邯郸淳《笑林》

      横无际涯。——宋· 范仲淹《岳阳楼记》

      横有八荒。——清· 梁启超《饮冰室合集·文集》

      横卧一楫。——明· 魏学洢《核舟记》

      又如:横头(在方桌筵席上,主席两侧的位置);横川(数字三的隐语);横字(横行的字);横阵(横排的阵势)

    4. 广,宽广

      广为横。——《史记·周本纪》正义

      又如:横广(广阔;广大);横沙(广阔的沙漠);横制(广泛控制);横览(广泛观察);横赐(帝王对臣民广施赏赐)

    〈动〉

    1. 横陈

      有神十人,各曰女娲之肠,横道而处。——《山海经·大荒西经》

      又如:横刀(横陈佩刀);横天(横陈天空);横身(插身,插手);横江(横陈江上);横路(横陈道中);横经(横陈经籍)

    2. 横渡;横越

      乃帅群臣横大河。——《汉书·扬雄传上》

      又如:横厉(横越;横渡);横截(横渡);横天(横越天空);横江(横越江上);横绝(横越;横度);横霄(横越天空)

    3. 充满;遮盖

      以横于天下。——《礼记·孔子闲居》。注:“充也。”

      横被四表。——《后汉书·冯异传》

      号以立横。——《礼记·乐记》

      又如:横世(充满世间);横出(充分表露;洋溢);横流(充盈;遍布);横滋(四处滋长)

    4. 下决心不顾一切 [steel one's heart]。

      如:横劲(决心,毅力);横了心(下定决心,不顾一切);横下心(犹横了心)

    5. 交错,错杂

      禁野之横行径逾者。——《周礼·野庐氏》。注:“妄行田中。”

      又如:横飞(交错飞行);横落(交错落下)

    6. 将物体横向拿着 。

      如:把扁担横过来;横刀跃马

    7. 横佩带着 。

      如:腰横玉带;横金妇(官员的妻子。横金:宋时表示官阶的一种佩带物)

    〈副〉

    1. 〈方〉∶大概,揣测之词 。

      如:横是(大概;恐怕)

    2. 〈方〉∶横竖,反正 。

      如:我横不那么办;横数(不管怎么样)

    3. 另见 hèng

    百科释义

    报错
    横是形声字，木为形，黄为声。本意是门前的栅栏或横木，引申为横向，并由此引申为纵横交错、成为横状、横着的意思，用作动词，横与宽、广相通时用作形容词引申指宽阔、广远，又引申为充溢。横又读作hèng，指不顺从、蛮横，又表示意外的。查看百科
